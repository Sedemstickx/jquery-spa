Welcome to SPA no hash(#) with JQuery

Hi guys, this is a simple and light(less than 70 lines of code as of now) Single Page Application (SPA) written in jquery that I decided to create for myself and developers that just learnt javascript and jquery and want to just build a simple SPA site e.g a portfolio website that will contain pages like about and contact or any site that wants to have that native app feel without having to go through the hustle of learning heavy frameworks just to be able to do same although for the future that's the way to go.

Why jquery and not just vanilla js?

Jquery because it's cross browser and you write less code whiles doing more.
